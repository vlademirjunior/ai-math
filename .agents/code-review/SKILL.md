# Code Review Skill

## Description
This skill provides a structured code review assistant. It analyzes code changes and offers suggestions for correctness, readability, maintainability, and security.

## Inputs
- `diff`: The code diff or file content to review.
- `context`: Optional project-specific notes or coding standards.

## Output
A structured response containing:
- Summary of findings
- Recommendations (fixes, improvements)
- Risk assessment

---

### Example Prompt
```
Review the following diff and provide a detailed code review:

<diff here>
```
