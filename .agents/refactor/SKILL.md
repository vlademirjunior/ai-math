# Refactor Skill

## Description
This skill helps refactor code by proposing concrete transformations that improve readability, reduce complexity, and increase testability.

## Inputs
- `code`: The current code snippet to refactor.
- `goals`: Refactor goals such as "reduce cyclomatic complexity" or "extract helper functions".

## Output
A refactored code suggestion with explanations and steps.

---

### Example Prompt
```
Refactor the following function to make it more modular and easier to test:

<code here>
```
