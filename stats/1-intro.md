# A estatística é a ciencia que permite obter fatos a partir de dados, ela envolve a coleta, organização e analise das informações

O Método estatístico é composto de etapas importantes:

1. Coleta de dados (amostragem ou censo)
2. Organizar em tabelas ou diagramas para que seja possível uma análise.
3. Apresentaçào dos dados para análise através de gráficos estatistico.
4. Análise de dados coletados (a matemática aqui)
5. Inferencias e conclusão

## Coleta de dados

- Variáveis estatísticas: há dois tipos mais comuns.

1. Variáveis QUALITATIVAS -> descrevem uma característica ou qualidade particular. => São divididas em categorias.
Exemplos: tipos de transporte, cor dos olhos, candidatos em uma eleição etc...

2. Variáveis QUANTITATIVAS -> Possuem valores numéricos. São divididas em discretas ou contínuas. e elas podem ser de 2 tipos, Discretas ou Continuas:

2.1: Variáveis quantitativas discretas: são valores exatos, que geralmente resultam de uma contagem. Ex.: O número de pessoas que moram em uma residencia, o número de carros de uma familia, número de lampadas com defeito, etc.

2.2: Variáveis continuas: assumem quaisquer valores em um intervalo e geralmente resultam de uma medição. Ex.: A altura de um prédio, o peso de bebes recém-nascidos, o tempo de vida útil de baterias de celular, etc...

## Amostragem e Censo

Censo: envolve a coleta de dados de 100% de individuos da população. O censo é detalhado e o método que produz resultados mais próximos da realidade, porém é custoso e muito demorado, logo não é prático. (não é o mais escolhido)

Amostragem: envolve a coleta de dados de apenas parte da população, a Amostragem deve ser feita de forma representativa e em quantidade suficiente para minimizar erros.

A amostra é um subconjunta de uma população. Deve representar bem uma população.

## Organização e Apresentação de dados

Utilizam-se gráficos estatísticos e tabelas de frequencia.

- Grafico de barras, Grafico de setores, diagrama de extremos e quartis.
- Tabelas de frequencia e diagramas de caule e folhas.

## Fluxo

População -> Amostragem (Amostra) -> descrição (Análise) -> conclusões/inferencias (População)
